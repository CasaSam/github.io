# Casa‑Sam — Sitio para GitHub Pages

Instrucciones rápidas para publicar:

1. Crea un repositorio nuevo en GitHub:
   - Para que la URL sea https://casasam.github.io/ crea un repo con el nombre `CasaSam.github.io`.
   - Si prefieres un proyecto con URL https://casasam.github.io/casa-sam crea `casa-sam` y activa GitHub Pages en Settings → Pages.

2. Añade los archivos:
   - En el repositorio, pulsa "Add file" → "Create new file".
   - Crea `index.html` y pega el contenido del archivo provisto.
   - Crea `styles.css` y pega el CSS.
   - Commit a la rama `main` (o `master` si así lo prefieres).

3. Publicar:
   - Si el repo se llama `CasaSam.github.io` se publicará automáticamente en minutos.
   - Si es un repo `casa-sam`, ve a Settings → Pages y selecciona la rama `main` y la carpeta `/ (root)` o `/docs`, guarda.

4. Editar:
   - Puedes editar los archivos desde el navegador (botón "Add file" → "Edit file" o presiona `.` para abrir el editor web tipo VS Code).
   - Cada commit actualiza la web automáticamente.

5. Formulario de contacto:
   - El formulario usa Formspree. Regístrate gratis en https://formspree.io, crea un formulario y copia el ID (reemplaza YOUR_FORM_ID en index.html).
   - Alternativa: usar un correo tipo contacto@gmail.com y enlazar `mailto:` si no quieres formulario.

6. Siguiente pasos (opcionales y gratis):
   - Añadir imágenes en carpeta `/images`.
   - Cambiar textos, colores (archivo `styles.css`).
   - Añadir un favicon (archivo `favicon.ico` y un `<link rel="icon" href="/favicon.ico">` en el head).

Si quieres, yo te doy:
- Las instrucciones paso a paso para crear el repositorio desde la interfaz de GitHub (con capturas de pantalla en texto).
- El contenido listo en un único bloque para copiar y pegar.
- O puedo generarte variantes (una plantilla más "portfolio", una con galería de imágenes, o una con soporte para varias páginas).
